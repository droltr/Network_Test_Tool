# Reporting & Exporting

Network Test Tool allows you to document network states and test results easily.

## ðŸ“„ Generating a Full Report
On the **Network Status** tab, click the **"Generate Report"** button.
*   **Format**: Human-readable Text file (`.txt`).
*   **Content**:
    *   System Information (Hostname, OS).
    *   Network Interface details (IP, MAC, Status).
    *   Current Diagnostics (Issues found).
    *   ARP Table summary.
    *   Connection statistics.
*   **Filename**: Automatically named with a timestamp, e.g., `network_report_20251224_203000.txt`.

## ðŸ’¾ Exporting Specific Test Results
Each tool (Ping, Port Scanner, Speed Test, Troubleshooter) has its own **Export** button.
*   These save the specific log of that tool to a text file.
*   Useful for saving a specific trace route or port scan result for later analysis.

## ðŸ”’ Privacy
*   Reports are saved locally on your computer.
*   No data is sent to the cloud or any external server.
