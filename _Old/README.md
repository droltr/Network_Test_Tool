# Network Tools

This project is a network issue detection tool that provides a modern graphical user interface (GUI) for users to diagnose and troubleshoot network problems on their computers.

## Features

- **Network Status Monitoring**: Displays the current network status and updates it based on network checks.
- **Ping Test**: Allows users to perform ping tests to check the connectivity to a specific IP address or domain.
- **Port Scanner**: Scans for open ports on a specified IP address, helping users identify potential vulnerabilities.
- **Speed Test**: Measures internet speed and displays the results in an easy-to-read format.

## Installation

To install the required dependencies, run the following command:

```
pip install -r requirements.txt
```

## Usage

To start the application, run the following command:

```
python src/main.py
```

This will launch the GUI, where you can access all the features mentioned above.
