# Kullanıcı İstekleri

## Network Status Ekranı
- başladığı andan itibaren otomatik olarak auto refresh etsin
- local network bağlı olmasına rağmen disconnected görünüyor, refresh etmeme rağmen düzelmedi
- IP adresi görünmüyor. (hala sorunlu)
- Cihazda birden fazla IP adresi varsa bunlar da tespit edilip belirtilsin. (başlamadı)
- Gateway görünmüyor. (Hala sorunlu - "Unable to determine" veya boş)
- DNS görünmüyor. (ip olarak değil, name olarak görnüyor, alt alta yazılmasını bekliyorum.)
- Connection Status düzgün çalışmıyor. (düzeldi, local network hatalı görünüyor)
- MAC adres görünmüyor. (düzeldi, küçük harflerle yazılsın, ekranda çok yer kaplıyor)
- Birden fazla ip adresi varsa, alt alta sıralasın, hangi network cihazı olduğu belirtilsin. (Hala sorunlu)
- Connection status kısmında hata varsa ":" sonrasında gelen kelime kırmızı olsun. (Çözüldü - Metin düzgün görünüyor, renk test edilecek)
- Ağ kartlarını disable/enable edebileceğim bir buton ekle.
